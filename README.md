# Dealora

A Vercel-ready MVP for analyzing short-term rental deals.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel

1. Create a GitHub repo and upload these files.
2. Create a free Vercel account.
3. In Vercel, click **Add New > Project**.
4. Import the GitHub repo.
5. Click **Deploy**.

No environment variables are required for this MVP.

## What this version includes

- STR deal analyzer
- saved scenarios in browser local storage
- comparison chart
- JSON export
- pricing tab
- simple beta lead capture stored locally
